```
unorderd_map(int, int)

for (a : A) {
  for (b : B) {
    map[a+b] ++;
  }
}

for (c : C) {
  for (d : D) {
    target = 0 - (c+d);
    if (map.find(target) != map.end) {
      count += map[target];
}
}
}
return count;
```
